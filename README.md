# GameClient

Javascript client for the Triangles In Space game. The game is a 3d multiplayer space shooter.

# Get Started
 - `git clone git@github.com:ZackMattor/tis-client.git`
 - `npm install`
 - `npm run dev`

# Stack
 - Node
 - ThreeJS
 - ParcelJS
